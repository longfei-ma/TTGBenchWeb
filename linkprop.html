Dataset FOOD (Leaderboard):

Temporal Graph: The FOOD dataset contains recipe details and reviews from Food.com (formerly GeniusKitchen). The FOOD dataset comprises a rich collection of cooking recipes and review texts, capturing 18 years of user interactions. This dataset provides insights into culinary trends, user preferences, and the evolution of recipes over time [1]. Each entry includes detailed recipe descriptions, ingredient lists, and user-generated reviews, making it an invaluable resource for studying temporal dynamics in food-related interactions. In this graph, users and recipes are denoted as nodes and a timespamped link between them is created if the user remarked the recipe at timestamp t. Each recipe has described texts of cooking steps and culinary category.

Prediction task: The considered task for this dataset is to predict whether a user will review a given cook recipe at a given time.

References

[1] Generating Personalized Recipes from Historical User Preferences Bodhisattwa Prasad Majumder*, Shuyang Li*, Jianmo Ni, Julian McAuley EMNLP, 2019.

Dataset IMDB (Leaderboard):

Temporal Graph: The IMDB dataset is composed of users, items and the reviews between them. Each movie has plot summary and the reviews are users' comment on there feelings [2]. Each review owns both raw textual contents and timestamp this review is conducted. The movies all prodive multile style tags, which we adopt as the temporal interests labels of users when a user comment this movie at some time. This dataset offers a detailed view of cinematic trends and audience perceptions over time.

Prediction task: The considered task for this dataset is to predict whether a user will review a given movie at a given time.

References

[2] Misra, Rishabh. "IMDB Spoiler Dataset." DOI: 10.13140/RG.2.2.11584.15362 (2019).

License: MIT license

Dataset Librarything (Leaderboard):

Temporal Graph: The Librarything dataset comes from a book review website, LibraryThing. The LibraryThing dataset focuses on book reviews, featuring detailed descriptions of books alongside user-generated reviews [3] [4]. This dataset captures a diverse range of literary works, from classic novels to contemporary bestsellers, and provides insights into reader preferences and trends over time. In this create graph, users and books are denoted as nodes and the timestamped reviews are created as temporal links among them with raw textural comments.

Prediction task: The considered task for this dataset is to predict whether a user will comment a given book at a given time.

References

[3] SPMC: Socially-aware personalized Markov chains for sparse sequential recommendation. Chenwei Cai, Ruining He, Julian McAuley. IJCAI, 2017
[4] Improving latent factor models via personalized feature projection for one-class recommendation. Tong Zhao, Julian McAuley, Irwin King. Conference on Information and Knowledge Management (CIKM), 2015.

Dataset Beeradvocate (Leaderboard):

Temporal Graph: The Beeradvocate dataset is dedicated to beer reviews, encompassing a wide variety of beers and user-generated ratings [5] [6]. We represent users and beer as the nodes of the temporal graph. The beers each owns styles, which can be assigned as temporal interests of users. This dataset offers a unique perspective on consumer preferences in the craft beer industry, documenting how user reviews and ratings have changed over time.

Prediction task: The considered task for this dataset is to predict whether a user will comment a given beer at a given time.

References

[5] Learning attitudes and attributes from multi-aspect reviews. Julian McAuley, Jure Leskovec, Dan Jurafsky. International Conference on Data Mining (ICDM), 2012
[6] From amateurs to connoisseurs: modeling the evolution of user expertise through online reviews. Julian McAuley, Jure Leskovec. WWW, 2013.

License: MIT license

Dataset Ratebeer (Leaderboard):

Temporal Graph: The Ratebeer dataset is another comprehensive collection of beer reviews, similar to the BeerAdvocate dataset but distinct in its user base and focus. This dataset documents a wide array of beers and includes detailed user ratings and reviews [5] [6], allowing to explore consumer behavior in the beer industry. The temporal aspect of the dataset enables investigations into how beer trends have evolved, the influence of marketing and branding on user perceptions, and the role of social interactions in shaping the craft beer community.

Prediction task: The considered task for this dataset is to predict whether a user will comment a given beer at a given time.

References

[5] Learning attitudes and attributes from multi-aspect reviews. Julian McAuley, Jure Leskovec, Dan Jurafsky. International Conference on Data Mining (ICDM), 2012
[6] From amateurs to connoisseurs: modeling the evolution of user expertise through online reviews. Julian McAuley, Jure Leskovec. WWW, 2013.

Dataset Amazon-Kindle (Leaderboard):

Temporal Graph: The Amazon-Kindle dataset is an Amazon online shopping dataset collected from [7] and is specifically focused on user reviews of various products in the Kindle category. This graph is a product review network spanning from 1996 to September 2023, composing of rich textural informations. Specifically, the products own rich descriptions and the reviews have rich rating contents. This dataset captures the evolution of consumer feedback, providing a rich source of information on user experiences with e-books and Kindle devices.

Prediction task: The considered task for this dataset is to predict whether a user will comment a given product at a given time.

References

[7] Yupeng Hou, Jiacheng Li, Zhankui He, An Yan, Xiusi Chen, and Julian McAuley. 2024. Bridging Language and Items for Retrieval and Recommendation. arXiv preprint arXiv:2403.03952 (2024).
