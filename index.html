TTGBench

Benchmark datasets for text-attributed temporal graph machine learning with Large Language Models

TTGBench is a collection of realistic, challenging and diverse scales benchmark datasets for text-attributed temporal graph machine learning. TTGBench includes both future link prediction and dynamic node classification tasks under two LLMs integration paradigms: LLM-as-Enhancer (LLM-generated text embeddings for temporal graph models), and LLM-as-Predictor (LLMs directly performing predictions via ICL or SFT). TTGBench is driven by community feedback and suggestions, if you would like to contribute a novel dataset or report any issues, please email us or visit our github.

Realistic datasets

TTGBench provides a diverse set of challenging and realistic benchmark datasets that are of varying sizes and support two core temporal graph machine learning tasks, including prediction of future link connection and dynamic node labels.

LLM-Centric methods

TTGBench systematically benchmarks 15 approaches under LLM-Centric paradigms: LLM-as-Enhancer (LLM-generated text embeddings for temporal graph models) and LLM-as-Predictor (LLMs directly performing predictions).

Unified evaluation

TTGBench provides standardized dataset splits and evaluators that allow for comparison of different models within a unified framework. TTGBench uses leaderboards to keep track of the state-of-the-art.
